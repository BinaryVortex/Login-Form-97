# Login Form

A clean, responsive login form built using HTML, CSS and a touch of JavaScript.

![Login Form Screenshot](./Screenshot%202024-08-06%20091131.png)

## Overview

This repository contains a simple, modern login page that demonstrates layout and styling using semantic HTML and CSS. It's intended as a small UI component you can reuse or learn from.

Key points:
- Mobile-friendly, centered login card
- Clear input focus and button states
- Lightweight: no build tools required

## Demo / Preview
Open `index.html` in your browser to view the form locally.

## Files
- `index.html` — markup for the login form
- `style.css` — visual styles and layout
- `Screenshot 2024-08-06 091131.png` — screenshot shown above

## How to use
1. Clone or download this repository.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
3. Optionally edit `style.css` to customize colors, spacing, or fonts.

No server or build step is required.

## Customization tips
- Change colors: edit the variables or color values in `style.css`.
- Add validation: add custom JS or use browser built-in form validation attributes.
- Replace background image: swap the `body` background or the included `fantasy-style-entryway-door-with-desert-landscape.jpg` if present.
